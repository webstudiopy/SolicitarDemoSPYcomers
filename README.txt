SPYComers Landing Pro

Cómo usar:
1. Descomprimí el ZIP.
2. Abrí el archivo index.html en tu navegador.
3. Si querés subirlo a Vercel o Netlify, subí toda la carpeta completa.

Archivos:
- index.html
- styles.css
- script.js
- assets/logo-spycomers.jpg

Esta versión usa colores blanco, azul y gris con estilo profesional.
